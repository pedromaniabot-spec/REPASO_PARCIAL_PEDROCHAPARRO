Acerca del sitio

Esta página contiene información general de ejemplo. Su objetivo es mostrar cómo se estructuran los contenidos en HTML con encabezados, párrafos y listas.

Datos interesantes

  • HTML significa HyperText Markup Language.
  • CSS permite dar estilo y color a las páginas web.
  • El navegador interpreta ambos para mostrar una interfaz atractiva.
